# Project-1-storm-insights

## First project: 
An analysis of severe weather events using the Preliminary Severe Storm Reports from NOAA dataset. This project explores weather patterns using SQL, focusing on hail storms, high winds, and tornados.

## Tools Used:
- BigQuery
- SQL
- Data cleaning and aggregation techniques

## Key Questions:
- What are the most frequently reported storm event types?
- How do severe storm events vary by year?
- Which states report the highest number of severe storm events?
- What are the average hail size and wind speed by storm type?
- How does the frequency of tornado reports compare across seasons?

## Insights
Hail events are the most frequently reported, suggesting they should be priorotized in resource allocation and ready planning. Most of the severe weather is localized in Texas, with the states directly north and in the cented of the USA getting their share of severe weather, with Kansas, Nebraska and South Dakota ranking as the next top three states to be prepared for adverse storms.
